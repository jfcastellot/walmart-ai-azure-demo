# Descripción de Clusters de Tiendas Walmart

## Cluster 0 — Tiendas de Alto Volumen

Estas tiendas tienen ventas promedio semanales superiores a $20,000 USD 
y un tamaño mayor a 150,000 ft². Se caracterizan por:

- Alta rotación de inventario
- Mayor sensibilidad a promociones y días festivos
- Ubicadas generalmente en zonas urbanas de alta densidad poblacional
- Requieren reabastecimiento más frecuente
- Mayor variabilidad en ventas durante temporadas especiales

## Cluster 1 — Tiendas de Bajo Volumen

Estas tiendas tienen ventas promedio semanales inferiores a $10,000 USD
y un tamaño menor a 100,000 ft². Se caracterizan por:

- Menor variabilidad en ventas semana a semana
- Menor impacto de eventos promocionales
- Ubicadas generalmente en zonas suburbanas o rurales
- Inventario más estable y predecible
- Mayor lealtad de clientela local

## ¿Cómo usar esta información?

Los gerentes de categoría pueden usar la segmentación para:
- Diseñar estrategias de promoción diferenciadas por cluster
- Optimizar niveles de inventario según el comportamiento del cluster
- Identificar tiendas atípicas que no encajan en ningún cluster
